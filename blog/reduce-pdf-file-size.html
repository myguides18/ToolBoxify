How to Reduce PDF File Size: The Ultimate Guide for 2026

A 40 MB PDF won't attach to an email, chokes slow connections, and ranks poorly when it's a downloadable resource on your site. Knowing how to reduce PDF file size is a basic digital skill — and in 2026 it takes seconds, not expensive software. This guide covers every lever, from quick wins to deep optimization.

Why PDFs Get So Big

PDFs balloon for three reasons: embedded images at full resolution, embedded fonts (sometimes the entire font file), and unnecessary metadata or layers. Identify the culprit and you know which fix to apply.

Lever 1: Compress Embedded Images

Images are almost always the heaviest part. Downsampling a 300 DPI scan to 150 DPI and re-encoding as JPEG can cut a file by 80% with no visible change on screen. This is the single biggest win for scanned documents and image-heavy reports.

Image DPI Cheat Sheet

  • Screen-only PDF: 72–96 DPI is plenty.
  • Print-quality: 150–200 DPI.
  • Archival scan: 300 DPI, but compress with JPEG regardless.

Lever 2: Subset and Strip Fonts

Embedding a full font (often 100–500 KB each) for a 2-page memo is wasteful. Font subsetting embeds only the characters you used. Most optimizers do this automatically; just make sure the option is on.

Lever 3: Remove What You Don't Need

  • Hidden layers and comments from review cycles.
  • Embedded JavaScript and form scripts you no longer use.
  • Document metadata and previous-version bookmarks.
  • Duplicate images repeated across pages.

How Small Is "Small Enough"?

Use caseTarget size
Email attachmentUnder 10 MB (ideally < 5 MB)
Web downloadUnder 3 MB per 20 pages
Mobile shareUnder 1 MB when possible

Common Mistakes

  • Over-compressing text scans until OCR becomes unreadable.
  • Forgetting that "max" compression can blur fine print.
  • Re-compressing an already-compressed PDF (diminishing returns, more artifacts).
