Answers to common questions about ToolBoxify and its browser-based tools.

General

Are my files uploaded?

No. The tools are designed to process selected files locally in your browser session.

Is ToolBoxify free?

Yes. ToolBoxify is free to use. Advertising may be used to support maintenance and hosting.

Does it work on mobile?

Yes, the site is responsive. Large PDFs and high-resolution batches may work better on desktop because they require more memory.

Images

Which image formats are supported?

The current tools support PNG, JPEG, and WEBP for compression and conversion.

Why did my file get larger?

Some images are already optimized. Converting formats or preserving detail can sometimes increase output size.

PDFs

Can I reorder PDFs before merging?

Yes. After selecting files in Merge PDF, drag the file rows to set the final order.

How do page ranges work?

Use ranges such as 1-3, 5, 7-9. The extracted file keeps the selected pages in ascending order.
